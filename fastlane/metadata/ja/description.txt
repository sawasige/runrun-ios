RunRunは、あなたが普段使っているランニングアプリと連携して、月間記録をフレンドと共有・競争できるアプリです。

Apple Watch、Strava、Nike Run Clubなど、ヘルスケア連携アプリで記録したランニングが自動で同期されます。

【主な機能】
・月別ランニング記録の表示
・フレンドとの月間走行距離ランキング
・タイムラインでフレンドの活動をチェック
・ペースに応じたルートのグラデーション表示

【対応アプリ】
・Apple Watch ワークアウト
・Strava
・Nike Run Club
・その他ヘルスケア連携アプリ

【こんな人におすすめ】
・ランニングを習慣化したい方
・仲間と一緒にモチベーションを高めたい方
・月間の走行距離を振り返りたい方

フレンドを追加して、一緒に走る楽しさを見つけましょう。
