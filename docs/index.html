RunRun ランニングノート

今月どれだけ走った？

Apple Watch、Strava、Nike Run Clubなど、普段のアプリで走った記録を自動取得して、シンプルな画面で振り返ろう。もちろん広告なし。

記録を、美しく書き出し。

ランの記録を画像や動画で書き出し。HDR対応で、対応デバイスでは映える仕上がりに。

背景はお気に入りの写真・動画から選択。SNSへの共有もワンタップ。

ペースで彩る、ルート。

GPSルートをペースに応じたグラデーションで表示。坂道や全力区間がひと目でわかる。

走った場所の地名もタイムラインや詳細に表示。

積み重なる、毎月・毎年。

月ごと、年ごとに積み重なる記録。自分の成長が、ひと目でわかる。

目標距離を設定すれば、達成率のグラフで習慣化をサポート。
