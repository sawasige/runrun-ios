よくある質問

ランニング記録が表示されません

以下をご確認ください:

  • ヘルスケアアプリでRunRunにデータへのアクセスを許可していますか？（設定 → ヘルスケア → データアクセスとデバイス → RunRun）
  • Apple Watch、Strava、Nike Run Clubなどでランニングを記録していますか？
  • アプリを下に引いてリフレッシュしてみてください

ランキングに表示されません

ランキングに表示されるには、データの同期が必要です。設定画面の「再同期」ボタンをお試しください。

アカウントを削除したい

アプリ内の「設定」→「退会する」から削除できます。アカウントを削除すると、サーバーに保存されたすべてのデータが完全に削除されます。
